# Tetris Game in Python

A simple Tetris game built using Python and the Tkinter library.

This project recreates the classic Tetris game where players move and rotate falling blocks to complete lines and earn points. It was created as a learning project to practice Python programming, game logic, and GUI development.

## About the Project

This project helped me learn:

* Python programming
* Object-Oriented Programming (OOP)
* Game development fundamentals
* Collision detection and movement logic
* Working with the Tkinter GUI library
* Using Git and GitHub for version control

## Features

* Classic Tetris gameplay
* Multiple Tetromino shapes
* Piece rotation
* Line clearing system
* Score tracking
* Game Over detection
* Built entirely with Python and Tkinter

## Technologies Used

* Python
* Tkinter

## Requirements

* Python 3.x

No additional libraries are required because Tkinter is included with most Python installations.

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/tetris-game.git
```

Navigate to the project folder:

```bash
cd tetris-game
```

## Running the Game

Run the following command in the project directory:

```bash
python tetris.py
```

If your system uses the `py` launcher:

```bash
py tetris.py
```

## Controls

| Key         | Action       |
| ----------- | ------------ |
| Left Arrow  | Move Left    |
| Right Arrow | Move Right   |
| Down Arrow  | Move Down    |
| Up Arrow    | Rotate Piece |
